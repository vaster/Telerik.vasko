Instructions

1. Add picture from the appbar -> .

Supported file formats: .jpg, .bmp,.gif ,.png .

2. Press the proccess button from the appbar - > .

Depending on the image quality and number of charachters, the time you're going to wait my escalate.

3. When the proccess finish, you can save your result from the save button from the appbar -> .

Supported file formats: .txt, .pdf.

(See the settings button in the appbar for more information about Auto Save Mode.)

Auto saving mode

Destination Folder: \Pictures\SimpleOCR

File type extension

.txt .pdf

(Imortant: Auto saving mode allows automatic save of the result file when the process is complete in you Pictures Library with the file type you define. Name of the saved files will look like follows: Sample0, Sample1, Sampl2 ...)

Once the saving mode is On it will apply its funcionality whenever the program is started, until the mode is turned Off.
